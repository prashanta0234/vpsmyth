
## Purpose
Contains the **frontend dashboard**.

### Subfolders

#### `assets/`
- CSS, JS, images, icons
- Static resources

#### `pages/`
- HTML pages
- Dashboard layouts
- API integration

---

## Best Practices
- Keep UI lightweight
- Avoid business logic in frontend
- Only interact with backend APIs
- Use responsive design
- Optimize images and assets
- Use semantic HTML
- Follow accessibility guidelines
- Use consistent styling
- Keep code modular and reusable
