# Contributing to VPSMyth

Thank you for your interest in contributing to VPSMyth! We welcome contributions from everyone.

## How to Contribute

1. **Report Bugs**: If you find a bug, please open an issue on GitHub.
2. **Suggest Features**: Have an idea? Open an issue to discuss it.
3. **Submit Pull Requests**:
    - Fork the repository.
    - Create a new branch for your changes.
    - Ensure your code follows the project's coding standards.
    - Run tests before submitting.
    - Open a pull request with a clear description of your changes.

## Coding Standards

- Follow Go best practices.
- Keep functions small and modular.
- Write tests for new features.
- Document your code.
